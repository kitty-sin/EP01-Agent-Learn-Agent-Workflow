Claude基本功第9.5集

為什麼錄9.5集呢

因為第9集我在講supabase的時候

後來我自己在研究發現

firebase一樣可以做到很棒的效果

但是設定上比較麻煩

如果大家不嫌麻煩的話

裝firebase才是最佳選擇

今天就跑一遍給大家看

一樣在我的Github影片底下

網址有這個懶人包

那你找到這一個4.5的懶人包

連接firebase資料庫

點MD檔點他在右上角

Download RAW file下載到你的Claude Code

開個New Session

將這個懶人包的MD檔拖到對話視窗

按提交接下來就會帶你一步步安裝firebase

只要裝一遍就好了

大家跟著我操作

接下來從頭開始跑懶人包

先執行步驟0環境檢查

檢查全部通過開啟瀏覽器

首先你要先註冊你的firebase

一般來說用你的Google帳號註冊就可以了

註冊好之後會到這個頁面來

接下來我們建立新的firebase專案

輸入專案名稱按繼續

Gemini in firebase這個不啟用按繼續

這項專案的Google分析也不啟用

按建立專案接下來按繼續

做完之後回到Claude Code裡面來

告訴他完成就會進行下一步

接下來手動操作照他的指示

在資料庫和儲存空間裡選擇FireStore

接下來按建立資料庫

這邊選擇Standard版本

下一步位置的話選最近的亞洲區

選臺灣這個按下一步

以正常版模式啟動按建立

完成之後會到這個頁面來

接下來照他的引導繼續設定

接下來要手動執行登錄

點選應用程式在上方搜尋CNB

找到命令提示字元點選

出現這個畫面接下來回到Claude Code

將這行指令點選複製

回到命令提示去貼上按Enter

在這邊的話選Yes再選Yes

接下來會引導你到達登錄帳號

選擇你的Google帳號登錄

允許好Firebase CLI Login完成

這件事你只要做一次就好了

告訴他完成任何有問題

有錯誤的地方Claude Code都會幫你修正

步驟三安裝Firebase MCP Server

好全部完成之後重新啟動Claude Code

接下來我們來測試一下完成

我們來看一下這是做好的即時文字雲

大家可以掃描這個QR Code來去試試看

完成之後我們回到Claude Code

告訴他在文字雲輸入文字

把資料抓出來看看完全沒問題

全部抓到了所以我們可以使用

自然語言來去操控我們的資料庫

未來我們使用AI去寫工具的時候

完完全全可以串接到我們自己的資料庫

Firebase相較於Superbase的話

更快容量更多也沒有時間限制

你可以告訴他移除Superbase的MCP

以後用不到了你就專心的使用Firebase就好了

好以上就是我們安裝比較麻煩的Firebase

但是功能比較強大我個人建議使用Firebase
